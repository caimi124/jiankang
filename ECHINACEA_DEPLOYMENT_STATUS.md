## 📞 部署问题排查

### 如果页面404
1. 清除浏览器缓存
2. 等待CDN刷新 (最多30分钟)
3. 检查 `lib/herb-detail-fallback.ts` 语法错误
4. 查看部署平台的构建日志

### 如果页面显示不完整
1. 检查浏览器控制台错误
2. 验证数据结构完整性
3. 检查前端组件是否正确渲染
4. 查看开发者工具 Network 标签

### 如果构建失败
1. 查看部署平台的构建日志
2. 本地运行 `npm run build` 测试
3. 检查 TypeScript 编译错误
4. 验证所有依赖项安装正确
