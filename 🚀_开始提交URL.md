## 🎯 **提交第1个URL：姜黄 (Turmeric)**

### 步骤：

1. **点击书签**
   - 在书签栏找到 `HerbScience - Google索引提交清单`
   - 展开 `第1天` 文件夹
   - 点击 **"1. 姜黄 Turmeric"**

2. **等待页面加载**
   - 自动跳转到Google Search Console
   - URL: `https://herbscience.shop/herbs/turmeric`
   - 页面会自动开始测试（等待10-30秒）

3. **查看测试结果**
   
   **可能显示以下之一**：
   
   #### 情况A：URL已收录
   ```
   ✅ URL已收录于Google服务中
   ```
   **操作**：仍然点击"请求编入索引"（刷新索引）
   
   #### 情况B：URL未收录
   ```
   ⚠️ URL未收录于Google服务中
   ```
   **操作**：点击"请求编入索引"（这就是我们的目标！）
   
   #### 情况C：URL在队列中
   ```
   ⏳ URL已在索引队列中
   ```
   **操作**：无需重复提交，继续下一个URL

4. **点击"请求编入索引"按钮**
   - 按钮位置：页面中间偏下
   - 等待5-10秒
   - 看到提示：✅ "已将URL加入编入索引队列"

5. **完成**
   - ✅ 第1个URL提交成功！
   - 关闭这个标签页
   - 继续第2个URL

## 💡 **提示技巧**

### 提速技巧：
1. **打开多个标签页**：一次打开3-4个书签，同时测试
2. **利用等待时间**：在一个URL测试时，准备下一个
3. **快捷键**：`Ctrl+W` 快速关闭标签页

### 常见情况处理：

#### 如果显示"配额已用尽"
- 说明今天已提交10-15个URL（达到上限）
- ✅ 停止提交
- ⏰ 明天继续剩余的URL

#### 如果页面加载失败
- 刷新页面重试
- 或稍等1分钟后再试
- 确保网络连接正常

#### 如果看不到"请求编入索引"按钮
- 可能这个URL最近刚提交过
- ✅ 直接跳过，继续下一个
- 📝 记录一下，几天后再检查
