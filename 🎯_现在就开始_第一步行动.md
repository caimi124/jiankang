## 📝 详细操作步骤 (选择了选项A或B的用户)

### Step 1: Google Search Console验证 (15分钟)

#### 方法1: HTML标签验证 (推荐)

1. **获取验证代码**
```bash
1. 访问 https://search.google.com/search-console/
2. 点击"添加属性"
3. 选择"网域"
4. 输入: herbscience.shop
5. 选择"HTML标签"验证方法
6. 复制verification code (类似: google1234567890abcdef.html)
```

2. **添加到网站**

打开 `app/layout.tsx`,找到第55-60行,修改为:

```typescript
verification: {
  google: 'YOUR_VERIFICATION_CODE_HERE', // ← 替换为GSC提供的代码
  other: process.env.BING_VERIFICATION_CODE ? {
    'msvalidate.01': process.env.BING_VERIFICATION_CODE
  } : undefined
},
```

3. **部署更新**
```bash
git add app/layout.tsx
git commit -m "Add Google Search Console verification"
git push origin main

# 等待Vercel自动部署 (2-3分钟)
```

4. **完成验证**
```bash
回到GSC,点击"验证"按钮
应该显示: "已验证" ✅
```

#### 方法2: DNS验证 (备选)

如果您管理域名DNS:
```bash
1. GSC选择"DNS记录"验证
2. 复制TXT记录值
3. 添加到域名DNS设置
4. 等待DNS传播 (5-30分钟)
5. 在GSC点击"验证"
```

## 📞 遇到问题?

### 常见问题解答:

**Q: GSC验证失败怎么办?**
```
A: 
1. 确认代码正确添加到app/layout.tsx
2. 确认网站已部署更新 (访问网站查看页面源代码)
3. 等待5分钟后再次尝试验证
4. 如果仍然失败,尝试DNS验证方法
```

**Q: Sitemap提交失败?**
```
A:
1. 检查sitemap.xml是否可访问: https://herbscience.shop/sitemap.xml
2. 检查是否有语法错误
3. 确认robots.txt允许访问sitemap
```

**Q: 请求索引后多久能被收录?**
```
A:
- 快的话: 3-7天
- 正常: 1-4周
- 慢的话: 2-3个月
取决于网站权重和外部链接数量
```
